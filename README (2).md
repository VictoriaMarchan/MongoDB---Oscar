# Nomeados ao Oscar

Este repositório contém a base de indicados ao Oscar em formato **MongoDB**, destinada a treinar comandos CRUD e consultas agregadas.

## Sobre a base de dados

A base contém informações como:

* Nome do indicado
* Nome do filme
* Categoria
* Ano da cerimônia
* Se venceu ou não
